# StoryFlow Creator UX 명세

## UX 비전

**"누구나 단숨에 이해하고, 30분 안에 만족할만한 페이지를 만든다"**

복잡함은 적이다. 모든 기능은 설명 없이도 직관적으로 이해되어야 한다.

---

## 디자인 원칙

### 1. 제로 러닝 커브
- 설명서 없이 바로 사용 가능
- 모든 버튼/아이콘에 명확한 라벨
- 첫 화면에서 바로 "섹션 추가" 가능

### 2. 즉각적 피드백
- 모든 액션에 0.1초 이내 반응
- 실시간 프리뷰 (편집 즉시 반영)
- 로딩/저장 상태 명확히 표시

### 3. 실수 방지 & 복구
- 삭제 전 확인 다이얼로그
- Ctrl+Z로 되돌리기
- 자동 저장 (데이터 손실 방지)

### 4. 일관성
- 동일한 액션은 동일한 위치
- 색상/아이콘 의미 통일
- 폰트/간격 시스템 준수

## 디자인 레퍼런스

### suzag.com (스크롤 경험)
- 전체화면 섹션
- 부드러운 스크롤 전환
- 미니멀하고 고급스러운 타이포그래피
- 충분한 여백

### Canva (인터랙션)
- 드래그 앤 드롭 직관성
- 클릭으로 바로 편집
- 시각적 피드백 (호버, 선택)
- 사이드바 기반 편집

## 인터랙션 패턴

### 드래그 앤 드롭
| 대상 | 액션 | 결과 |
|------|------|------|
| 이미지 파일 | 드롭 영역에 드롭 | 이미지 업로드 |
| 섹션 | 위/아래로 드래그 | 순서 변경 |
| Gallery 이미지 | 드래그 | 순서 변경 |

### 호버 효과
| 요소 | 호버 시 |
|------|---------|
| 버튼 | 밝아짐 + 커서 포인터 |
| 섹션 카드 | 테두리 강조 |
| 삭제 버튼 | 빨간색 강조 |

### 키보드 단축키
| 단축키 | 기능 |
|--------|------|
| Ctrl+Z | 실행 취소 |
| Ctrl+Y / Ctrl+Shift+Z | 다시 실행 |
| Ctrl+S | 저장 |
| Delete | 선택 섹션 삭제 (확인 후) |

## 에러 처리 UX

### 친절한 에러 메시지
```
❌ "Error: Invalid file type"
✅ "이미지 파일만 업로드할 수 있어요 (PNG, JPG, GIF)"

❌ "Upload failed"
✅ "업로드에 실패했어요. 파일 크기가 5MB 이하인지 확인해주세요"

❌ "Network error"
✅ "인터넷 연결을 확인해주세요"

❌ "Authentication required"
✅ "로그인이 필요해요. Google 계정으로 로그인해주세요"
```

### 에러 표시 방식
- 토스트 알림 (3초 후 자동 사라짐)
- 인라인 에러 (입력 필드 아래)
- 모달 (중요한 확인 필요 시)

## 반응형 디자인

### 브레이크포인트
| 이름 | 너비 | 편집 패널 |
|------|------|-----------|
| Mobile | < 768px | 숨김 (프리뷰만) |
| Tablet | 768px ~ 1024px | 280px |
| Desktop | > 1024px | 320px ~ 384px |

### 모바일 대응
- 편집은 태블릿/데스크탑 권장
- 모바일은 프리뷰 모드만 지원
- 모바일 진입 시 안내 메시지

## 접근성

### 시맨틱 HTML
- `<section>` - 각 섹션
- `<nav>` - 네비게이션
- `<button>` - 모든 클릭 요소
- `<label>` - 입력 필드 라벨

### ARIA
- `aria-label` - 아이콘 버튼
- `aria-expanded` - 아코디언
- `role="alert"` - 에러 메시지

### 색상 대비
- 텍스트: 최소 4.5:1 대비
- 큰 텍스트: 최소 3:1 대비
- 버튼: 명확한 시각적 구분
