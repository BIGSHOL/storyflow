# StoryFlow Creator 확장 프로젝트 개요

## 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 프로젝트명 | StoryFlow Creator 확장 |
| 유형 | 기존 프로젝트 기능 확장 |
| 개발자 | 1인 개발 |
| 상태 | 기획 완료 |

---

## 비전

**"코딩 없이 누구나 쉽게, 30분 안에 만족할만한 스토리텔링 웹페이지 만들기"**

복잡함을 배제하고, 누구나 단숨에 이해할 수 있는 직관적인 노코드 웹 빌더를 만듭니다.

---

## 핵심 가치

1. **극도의 단순함** - 복잡한 서비스의 안티테제, 단숨에 이해 가능
2. **시각적 직관성** - Canva처럼 드래그 앤 드롭
3. **세련된 결과물** - suzag.com처럼 고급스러운 스크롤 경험
4. **안정성** - 버그 최소화, 신뢰할 수 있는 서비스

---

## 타겟 사용자

### 주요 페르소나
- **프리랜서** - 포트폴리오, 서비스 소개 페이지 필요
- **소상공인** - 간단한 홈페이지, 이벤트 홍보 페이지 필요

### 사용 시나리오
1. 서비스 홈페이지 제작
2. 포트폴리오 페이지 제작
3. 이벤트/프로모션 홍보 페이지

---

## 현재 상태 (As-Is)

### 구현된 기능
- 5종 레이아웃 (Hero, Split Left/Right, Image Overlay, Simple Text)
- 이미지/비디오 업로드 (클릭 + 드래그앤드롭)
- 실시간 프리뷰 (데스크탑/모바일)
- HTML 내보내기
- 로컬 저장/불러오기 (localStorage)
- Undo/Redo
- 자동 저장

### 한계점
- 레이아웃 종류 부족
- 백엔드 없음 (데이터 영구 저장 불가)
- 협업/공유 기능 없음

---

## 목표 상태 (To-Be)

### Phase 1: MVP - 추가 레이아웃
- Gallery (이미지 그리드)
- Timeline (시간순 스토리)
- Cards (카드형 나열)
- Quote (인용문 강조)
- Stats (숫자/통계 강조)
- Video Hero (비디오 배경)

### Phase 2: 백엔드 연동
- Supabase 연동
- Google 소셜 로그인
- 클라우드 미디어 저장 (이미지, GIF 중심)

### Phase 3: 협업 기능
- 링크 공유 (읽기 전용)
- 편집 권한 공유
- (향후) 프리미엄 기능 기반

---

## 성공 지표

| 지표 | 목표 |
|------|------|
| 페이지 제작 시간 | 코딩 모르는 사람 30분 이내 |
| 사용자 만족 | 10명 이상이 자신이 만든 페이지에 만족 |
| 안정성 | 버그 0에 가깝게 |

---

## 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | React 19 + TypeScript + Vite 6 |
| Styling | Tailwind CSS |
| Backend | Supabase (PostgreSQL + 실시간 + 인증) |
| Auth | Google 소셜 로그인 |
| Storage | Supabase Storage (이미지, GIF) |
| Hosting | 미정 (Vercel 추천, Firebase Hosting 경험 있음) |
