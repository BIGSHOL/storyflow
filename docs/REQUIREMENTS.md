# StoryFlow Creator 요구사항 명세

## 기능 요구사항

### Phase 1: 추가 레이아웃 (MVP)

#### FR-1.1 Gallery 레이아웃
- **설명**: 이미지를 그리드 형태로 표시
- **세부 요구사항**:
  - 2~4열 그리드 선택 가능
  - 이미지 클릭 시 라이트박스 확대
  - 이미지 간격 조절 가능
  - 캡션 표시 옵션
- **우선순위**: P0

#### FR-1.2 Timeline 레이아웃
- **설명**: 시간순 스토리를 세로 타임라인으로 표시
- **세부 요구사항**:
  - 좌/우 번갈아 배치 또는 한쪽 정렬
  - 날짜/제목/설명 입력
  - 연결선 스타일 커스터마이징
  - 아이콘 또는 이미지 마커
- **우선순위**: P0

#### FR-1.3 Cards 레이아웃
- **설명**: 카드형 콘텐츠 나열
- **세부 요구사항**:
  - 2~4열 카드 배치
  - 카드당 이미지 + 제목 + 설명
  - 호버 효과 옵션
  - 카드 스타일 (둥근 모서리, 그림자 등)
- **우선순위**: P0

#### FR-1.4 Quote 레이아웃
- **설명**: 인용문/강조 텍스트 전용
- **세부 요구사항**:
  - 큰 인용 부호 장식
  - 출처/저자 표시
  - 배경색/이미지 선택
  - 텍스트 정렬 및 크기 조절
- **우선순위**: P0

#### FR-1.5 Stats 레이아웃
- **설명**: 숫자/통계 강조 표시
- **세부 요구사항**:
  - 2~4개 숫자 카드 배치
  - 숫자 + 라벨 + 설명
  - 카운트업 애니메이션 옵션
  - 아이콘 추가 가능
- **우선순위**: P0

#### FR-1.6 Video Hero 레이아웃
- **설명**: 비디오 배경 전체화면 섹션
- **세부 요구사항**:
  - MP4/WebM 비디오 지원
  - 자동 재생, 음소거, 루프
  - 오버레이 + 텍스트
  - 폴백 이미지 (비디오 로드 실패 시)
- **우선순위**: P0

---

### Phase 2: 백엔드 연동

#### FR-2.1 Supabase 연동
- **설명**: 프로젝트 데이터 클라우드 저장
- **세부 요구사항**:
  - 프로젝트 CRUD (생성, 읽기, 수정, 삭제)
  - 섹션 데이터 JSON 저장
  - 프로젝트 목록 조회
  - 실시간 동기화 (선택적)
- **우선순위**: P1

#### FR-2.2 Google 소셜 로그인
- **설명**: Google 계정으로 간편 로그인
- **세부 요구사항**:
  - Supabase Auth + Google OAuth
  - 로그인/로그아웃 UI
  - 사용자 프로필 표시
  - 세션 유지
- **우선순위**: P1

#### FR-2.3 미디어 클라우드 저장
- **설명**: 이미지/GIF 클라우드 업로드
- **세부 요구사항**:
  - Supabase Storage 연동
  - 이미지: PNG, JPG, WebP (최대 5MB)
  - GIF: 최대 10MB
  - 비디오: 제한적 지원 (용량 고려)
  - 업로드 진행률 표시
- **우선순위**: P1

---

### Phase 3: 협업 기능

#### FR-3.1 링크 공유
- **설명**: 프로젝트 결과물 링크 공유
- **세부 요구사항**:
  - 고유 공개 URL 생성
  - 읽기 전용 뷰어 페이지
  - 공유 링크 복사 버튼
  - 공유 설정 (공개/비공개)
- **우선순위**: P2

#### FR-3.2 편집 권한 공유
- **설명**: 다른 사용자에게 편집 권한 부여
- **세부 요구사항**:
  - 이메일로 초대
  - 권한 수준 (보기/편집)
  - 협업자 목록 관리
  - 권한 취소
- **우선순위**: P2

## 비기능 요구사항

### NFR-1 성능
| 항목 | 목표 |
|------|------|
| 초기 로딩 | 3초 이내 |
| 섹션 추가 반응 | 100ms 이내 |
| 이미지 업로드 | 진행률 표시, 5MB 이내 2초 |
| 스크롤 | 60fps 유지 |

### NFR-2 사용성
| 항목 | 목표 |
|------|------|
| 학습 시간 | 5분 이내 기본 기능 파악 |
| 페이지 제작 | 30분 이내 완성 |
| 에러 복구 | 모든 에러에 친절한 안내 |

### NFR-3 안정성
| 항목 | 목표 |
|------|------|
| 버그 | 0에 가깝게 |
| 자동 저장 | 3초 디바운스 |
| 데이터 손실 | 0건 |

### NFR-4 호환성
| 항목 | 지원 범위 |
|------|----------|
| 브라우저 | Chrome, Firefox, Safari, Edge (최신 2버전) |
| 디바이스 | 데스크탑, 태블릿 (편집), 모바일 (뷰어) |
| 내보내기 | 모든 모던 브라우저에서 렌더링 |

## 제약 조건

### 기술적 제약
- 1인 개발 (복잡한 기능은 단계적 구현)
- 비디오 저장은 용량/비용 제한

### 비즈니스 제약
- 기본 무료 서비스
- 프리미엄 기능은 향후 검토

## 용어 정의

| 용어 | 정의 |
|------|------|
| 섹션 | 페이지를 구성하는 하나의 블록 단위 |
| 레이아웃 | 섹션의 시각적 배치 템플릿 |
| 프로젝트 | 여러 섹션으로 구성된 하나의 페이지 |
| 내보내기 | 프로젝트를 HTML 파일로 변환하여 다운로드 |
