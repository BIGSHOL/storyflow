# Integration Validator Command

StoryFlow Creator 코드 품질 및 통합 검증 커맨드입니다.

## 사용법

```
/validate
```

## 역할

코드 변경 후 전체 프로젝트의 품질과 정합성을 검증합니다.

## 검증 항목

### 1. 빌드 검증

```bash
npx tsc --noEmit  # TypeScript 타입 검사
npm run build     # Vite 프로덕션 빌드
```

### 2. UX 체크리스트

#### 노코드 사용성
- [ ] 첫 화면에서 바로 시작 가능
- [ ] 모든 기능이 마우스로 조작 가능
- [ ] 전문 용어 없이 이해 가능
- [ ] 에러 메시지가 친절함

#### 세로 스크롤 경험
- [ ] 부드러운 스크롤 동작
- [ ] 각 섹션이 적절한 높이 (min-h-screen)
- [ ] 첫 화면에 스크롤 힌트 표시
- [ ] 모바일에서도 정상 동작

### 3. 레이아웃 검증

- [ ] Hero 레이아웃 정상 렌더링
- [ ] Split Left 레이아웃 정상 렌더링
- [ ] Split Right 레이아웃 정상 렌더링
- [ ] Image Overlay 레이아웃 정상 렌더링
- [ ] Simple Text 레이아웃 정상 렌더링

### 4. 이미지 업로드 검증

- [ ] 파일 선택으로 업로드 가능
- [ ] 이미지 미리보기 표시
- [ ] 이미지 변경/삭제 가능
- [ ] 지원하지 않는 파일 에러 처리

### 5. 편집 기능 검증

- [ ] 제목/설명 텍스트 편집
- [ ] 레이아웃 변경
- [ ] 색상 변경 (배경/텍스트)
- [ ] 투명도 조절 슬라이더
- [ ] 섹션 순서 변경 (위/아래)
- [ ] 섹션 삭제

### 6. 프리뷰 기능 검증

- [ ] 실시간 프리뷰 반영
- [ ] 데스크탑/모바일 전환
- [ ] 전체화면 프리뷰 모드

## 검증 프로세스

```
/validate 실행
    ↓
[1] TypeScript 타입 검사
    ↓
[2] Vite 빌드 테스트
    ↓
[3] 체크리스트 항목 확인
    ↓
결과 보고
```

## 출력 형식

```
=== StoryFlow Creator 검증 결과 ===

✅ TypeScript: 에러 없음
✅ Vite Build: 성공

UX 체크리스트:
✅ 노코드 사용성 - 통과
✅ 세로 스크롤 경험 - 통과
⚠️ 드래그 앤 드롭 - 미구현 (P0)

레이아웃 검증:
✅ Hero - 정상
✅ Split Left - 정상
✅ Split Right - 정상
✅ Image Overlay - 정상
✅ Simple Text - 정상

기능 검증:
✅ 이미지 업로드 - 정상
✅ 텍스트 편집 - 정상
✅ 실시간 프리뷰 - 정상
⚠️ HTML 내보내기 - 미구현 (P0)

=== 검증 완료 ===
```

## 미구현 기능 현황

### P0 (필수)
- [ ] HTML 내보내기 기능
- [ ] 드래그 앤 드롭 이미지 업로드

### P1 (중요)
- [ ] 프로젝트 저장/불러오기 (localStorage)
- [ ] Undo/Redo

### P2 (선택)
- [ ] 섹션 드래그 순서 변경
- [ ] 이미지 크롭 기능
- [ ] 추가 레이아웃 (Gallery 등)
